#### 数据结构

从结构逻辑上分两种：线性结构、非线性结构

线性结构：是一个有序数据元素的集合。其中数据元素之间的关系是一对一的关系，即除了第一个和最后一个数据元素之外，其他数据元素都是收尾相接的。

常用的线性结构有：栈，队列，链表，线性表

非线性结构：各个数据元素不再保持在一个线性序列中，每个数据元素可能与另个或多个其他数据元素发生联系

常见的非线性结构有 二维数组，树等
