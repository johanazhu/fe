# 用Node自定义脚手架



我们要使用 node 来创建一个自己的 脚手架 工具，暂时起名为 azhu-cli



这里需要知道一些 shell 命令，如果你不懂也没关系





## 创建工程



```shell
mkdir azhu=cli
cd azhu-cli
npm init -y
```
