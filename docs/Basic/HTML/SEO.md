## SEO

### 哥飞

跟豆瓣学 SEO
首页一定要有一个 H1 标签，让搜索引擎知道网站的核心关键字是什么，豆瓣创造性的把 logo 图和 h1 结合到一起了。
内容页，网站名称就不能作为主题了，所以 H1 是书籍电影名称，同时每一个 H2 都重复主题。让搜索引擎知道，这个页面的内容都是跟主题相关的。
