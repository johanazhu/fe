# 立即执行函数的魅力



我们阅读源码时，常看到

```javascript
;(function() {
   'use strict' 
}())
// 或者
;(function() {
    'use strict'
})()
```



这种叫 IIFE 立即执行函数







### 升级

再来看看这个 `underscore` 的源码

```javascript
(function(global, factory) {
    ...
}(this, (function () {
    ...
})));
```

套中套，草他妈

其实不然，一个优秀的库之所以这么写，肯定是有原因，要找到其中的原因，而不是满嘴粗口
