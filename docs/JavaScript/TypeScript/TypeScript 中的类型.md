## 联合类型

类型注释

TypeScript 通过类型注解提供编译时的静态类型检查，可以在编译阶段就发现潜在 Bug，同时让编码过程中的提示也更智能。使用方式很简单，在 `:` 冒号后面注明变量的类型即可。

```typescript
const str: string = 'abc';
```
