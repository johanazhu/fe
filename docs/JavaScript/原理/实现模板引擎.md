# 实现模板引擎

模板引擎我们之前看到过很多，比如 hbs，template....

那么看他们的源码，对于新手来说并不友好，新手之所以为新手，在于他基础不行，看一些难一点的代码，就看不懂，所以我们做一件简单的，有利于新手理解的模板引擎，然后根据这个模板引擎来做一个简单的 demo，以此加深我们对模板引擎的理解

核心掌握，剩下的就是兼容和加各种好用功能的方法。
