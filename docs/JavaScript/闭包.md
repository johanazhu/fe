# 闭包

#### 闭包

什么是：即重用一个变量，又保护变量不被篡改的一个机制

为什么会有它

​	全局变量： 优：可重用

​						缺：造成全局污染，易被篡改

​	局部变量： 优：仅函数内可使用，不会污染全局，也不会被篡改

​						缺：不可重用

何时：只要希望重用一个变量，又保护变量不被污染
