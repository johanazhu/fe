# 函数式编程



要想看懂一些牛逼的源码

必须掌握的编程思维就是 函数式编程

```javascript
const compose = (f, g) => x => f(g(x))

const f = x => x + 1;
const g = x => x * 2'
const fg = compose(f, g)
fg(1)
```
