# 写在前面

打造知识体系，这不是件容易的事。在 19 年时，笔者看到不少同行在说一个词——**前端体系**。前端不单单只是 HTML、CSS、JavaScript、NodeJS 以及各种构建工具和三大框架，它更包括小程序、微信生态、跨平台、大前端概念等等。随着工龄的提升，知识面被不断打开，笔者也逐渐从一个小开发到能独单一面完成一项工作任务
