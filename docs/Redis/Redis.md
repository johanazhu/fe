# Redis

### 什么是Redis

​	开源

​	Redis 是 Remote Dictionary Server(远程数据服务)的缩写

​	Redis是 内存高速缓存数据库

​	数据模型为 key-value

​	多种数据结构： string，list  hash set sorted set

​	可持久化，保证了数据安全





Redis 常见命令行

| 命令名        | 描述          |
| ------------- | ------------- |
| keys *        | 查看所有的key |
| set key value | 设置key-value |
| get key       | 得到某个key   |
| del key       | 删除某个key   |
|               |               |
|               |               |
