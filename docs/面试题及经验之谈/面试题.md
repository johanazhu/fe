# 阿里笔试题：链式调用、对象比较以及对象是否存在循环引用

```javascript
for (var i = 0; i < 5; i++) {
    setTimeout(function () {
        console.log(i);
    }, 1000);
}
```

55555

为什么？

setTimeout 是异步，延迟 1 秒后执行，for 循环打印了 5 个 console.log(i)，但是 i 是全局变量，

```javascript
for (var i = 0; i < 5; i++) {
    (function (j) {
        setTimeout(function () {
            console.log(j);
        }, 1000);
    })(i);
}
```
