```javascript
for (var i=0; i<5; i++) {
    setTimeout(function() {
        console.log(i)
    }, 1000)
}
```

55555

为什么？

setTimeout 是异步，延迟1秒后执行，for循环打印了5个console.log(i)，但是i是全局变量，



```javascript
for (var i=0; i<5; i++) {
    (function(j) {
        setTimeout(function() {
            console.log(j);
        }, 1000);
    })(i);
}
```
