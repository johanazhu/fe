# 垂直居中

10种方式

##### 仅居中元素定宽高

​	absolute + 负margin

​	absolute + margin auto

​	absolute + calc

##### 居中元素不定宽高

​	absolute + transform

​	line-height	行内元素

​	writing-mode

​	table

​	css-table

​	flex

​	grid



css3 不能在IE9以下使用，移动端随便使用
