## 在浏览器地址栏输入 URL 到页面现实，浏览器到底发生了什么？

​ 从输入那一刻，我们会在浏览器本地内存中查数据，当它发送 HTTP 请求后，就与 HTTP 世界有了一层交流，在 HTTP 世界里，命令经过 DNS 解析，缓存，有可能还会进行 301 重定向。。。

​ 它们之间的世界如下所示：

​ 最后返回浏览器拿到 HTTP 世界返回的数据，通过渲染引擎渲染页面

​ 因为渲染加载能影响用户体验，所以这一块的回流与重绘要格外注意，同样，也是考点之一
