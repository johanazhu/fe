# FastAPI



一个用于构建 API的现代、快速(高性能)的web框架。



fastapi 是建立在Starlete和Pydantic基础上的，Pydantic是一个基于Pyhon类型提示来定义数据验证、序列化和文档的库。Starlete是
种轻量级的ASGI框架/工具包，是构建高性能Asyncio服务的理性选择。



Starlete 负责 web 部分(Asyncio)
Pydantic 负责数据部分(类型提示)
