
#### git branch

```shell
git branch 查看本地所有的分支
git branch -f 查看所有远程的分支
git branch -a 查看所有远程分支和本地分支
git branch -D <分支名> 删除本地分支
```



#### git checkout

如果文件还在工作区，还没添加到暂存区，可以使用git checkout撤销

```shell
git checkout [file] 丢弃某个文件file
git checkout . 丢弃所有的文件
```
