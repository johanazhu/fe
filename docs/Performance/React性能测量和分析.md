## FCP

首次内容绘制（FCP）指标测试页面从开始加载到页面内容的任何部分在屏幕上完成渲染的时间。对于该指标，”内容“指的是文本、图像（包括背景图像）、`<svg>`元素或非白色的`<canvas>`元素。
