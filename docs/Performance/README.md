### preload（预加载） 和 prefetch 的区别

preload 是页面加载前加载，

prefetch 异最低优先级 lowest 来提前加载

preload 加载资源一般是当前页面需要，它告诉浏览器预先请求当前页面需要的资源

prefetch 一般是其他页面需要用到的资源

defer 或者 async
