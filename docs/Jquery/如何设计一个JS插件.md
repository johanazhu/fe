# 如何设计一个 JS/jQuery 插件

## 前言

许多开源库都有插件系统，例如：Vue、JQuery、Gatsby。插件是库和框架的一个常见特性，用于运行开发人员以一种安全、可伸缩的方式添加工鞥呢。那么怎么构建一个插件系统？让我们用 JS 构建一个自己的插件系统来回答这个问题
