# 组件最佳实现





之前 React 的组件有好几种

- Functional Component
- PureComponent
- HOC
- Component

在没有Hook 时，

如果一个组件没有自身状态，应该使用函数式组件

如果组件是纯组件（属性都是简单值），那么应该使用 PureComponent

如果要复用同一逻辑，用HOC（有了Hook，也可以代替）

否则应该用 Component
