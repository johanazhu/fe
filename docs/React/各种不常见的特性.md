## Suspense

什么是 Suspense？

Suspense 其实在某个 API 被调用时也可以处理我们的加载状态

React Suspense 是组件从缓存中加载数据时暂停呈现的一种通行方式

它解决的问题：渲染是和 I/O 绑定时的情况
