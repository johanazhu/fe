# 关于我

笔者是个懒汉
又是个拖延症
因为面试时需要拿出点所谓的github star数，被迫思考要做点产品出来，19年跳槽到一家国企后，就开始酝酿做一款自己的产品
从 产品——设计——前端——后端——上线
想法总是丰满的，但是现实却很残酷。
不过还好，最后完成了

但是这个产品完成之前，笔者看到一部分开发人员将自己的网站做成知识图谱

笔者开始了边做产品边做知识图谱的苦逼生涯

此网站仅为笔者所收集和总结的一点点前端开发所需要掌握知识
